# ETHGlobal New York 2026 Notes

## Build Posture

This repository is the from-scratch public submission repository for DreamNet Quorum Lab.

The broader DreamNet universe influenced the idea, language, and operator goals, but this repo is the standalone hackathon implementation. Do not copy private monorepo code into this repository.

## AI Attribution

AI tools may be used during the event for:

- Planning.
- UI implementation.
- TypeScript review.
- Copy editing.
- Test and demo-script generation.

Human responsibilities:

- Product direction.
- Final code approval.
- Video narration.
- Submission text.
- Any deploy or external mutation approval.

## Safety Boundaries

The project remains paper-mode only unless explicitly changed after the hackathon:

- No trading execution.
- No wallet signing.
- No fund movement.
- No public posting.
- No hidden autonomous side effects.

## Submission Checklist

- Public repository.
- Small commits during the event.
- Working local demo.
- Cloudflare-compatible deploy path.
- Two-to-four minute demo video.
- Clear explanation of what was built during the event.
